# GodotExample-2D

## Setup

Download dodge_the_creeps_2d_assets.zip from
https://docs.godotengine.org/en/stable/getting_started/first_2d_game/01.project_setup.html
and extract in the top game directory to get the directories
dodge_the_creeps_2d_assets/art and dodge_the_creeps_2d_assets/fonts.

Do not commit these downloaded files to this repository to avoid any licese issues.
